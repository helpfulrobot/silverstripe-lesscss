# LESS Module for SilverStripe CMS

Simple wrapper for [lessphp](http://leafo.net/lessphp/) to integrate [LESS](http://lesscss.org/) in SilverStripe.

## Requirements

 * Silverstripe 2.4

## Usage

In your Template.ss:

<pre>
	…
	&lt;% require css(themes/mytheme/css/styles.less) %&gt;
	…
</pre>
